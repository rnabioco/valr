Read a VCF file.

Usage

read_vcf(vcf)

Arguments

vcf
vcf filename

Value

data_frame

Description

Read a VCF file.

Note

return value has chrom, start and end columns. Interval lengths are the size of the 'REF' field.

Examples

v <- system.file('extdata', 'test.vcf.gz', package = 'valr')
read_vcf(v)
