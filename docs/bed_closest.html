Identify closest intervals.

Usage

bed_closest(x, y, overlap = TRUE, strand = FALSE, strand_opp = FALSE,
  suffix = c(".x", ".y"), distance_type = c("genome", "strand", "abs"))

Arguments

x
tbl of intervals
y
tbl of intervals
overlap
include overlapping intervals and report overlap
strand
intersect intervals on same strand
strand_opp
intersect intervals on opposite strands
suffix
colname suffixes in output
distance_type
reporting method for distance to nearest interval ( genome (default) = use negative distances to report upstream intervals, strand = define upstream based on strand, abs = report absolute value of distance)

Value

data_frame

Description

Identify closest intervals.

Examples

x <- tibble::frame_data(
~chrom, ~start, ~end,
"chr1", 500,    600,
"chr2", 5000,   6000
)

y <- tibble::frame_data(
~chrom, ~start, ~end,
"chr1", 100,    200,
"chr1", 150,    200,
"chr1", 550,    580,
"chr2", 7000,   8500
)

bed_closest(x, y)


# A tibble: 4 × 7
  chrom start.x end.x start.y end.y .overlap .distance
  <chr>   <dbl> <dbl>   <dbl> <dbl>    <int>     <int>
1  chr1     500   600     550   580       30         0
2  chr1     500   600     100   200        0      -300
3  chr1     500   600     150   200        0      -300
4  chr2    5000  6000    7000  8500        0      1000
