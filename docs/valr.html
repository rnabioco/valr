valr: genome interval arithmetic in R

Description

valr provides tools to read and manipulate intervals and signals on a genome reference. While other analysis suites enable command-line analysis, valr enables similar analyses within R.

Details

valr focuses on manipulating intervals in BED format and signal in bedGraph format. Eventually this package will power interactive visualizations of genome-scale data with shiny.

To learn more about valr, start with the vignette: browseVignettes(package = "valr")
