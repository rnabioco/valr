Identify intervals in a genome not covered by a query.

Usage

bed_complement(x, genome)

Arguments

x
tbl of intervals
genome
chrom sizes

Value

data_frame

Description

Identify intervals in a genome not covered by a query.

Examples

genome <- tibble::tribble(
   ~chrom,  ~size,
   "chr1", 500,
   "chr2", 600,
   "chr3", 800
)

x <- tibble::tribble(
   ~chrom, ~start, ~end,
   "chr1", 100,    300,
   "chr1", 200,    400,
   "chr2", 1,      100,
   "chr2", 200,    400,
   "chr3", 500,    600
)

# intervals not covered by x
bed_complement(x, genome)
