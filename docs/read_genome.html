read genome files (i.e., UCSC "chromSize" files)

Usage

read_genome(filename)

bound_intervals(x, genome, trim = FALSE)

Arguments

filename
file containing chrom/contig names and sizes, one-pair-per-line, tab-delimited
x
a tbl of intervals
genome
a tbl of chrom sizes
trim
adjust coordinates for out-of-bounds intervals

Value

data_frame with colnames chrom and size, sorted by size

data_frame

Description

Genome files contain chromosome name and size information. These sizes are used by downstream functions to identify computed intervals that have ends outside the defined size.

Select intervals bounded by a genome.

Examples

genome <- system.file('extdata', 'hg19.chrom.sizes.gz', package = 'valr')
read_genome(genome)
