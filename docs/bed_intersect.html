Identify intersecting intervals.

Usage

bed_intersect(x, y, invert = FALSE, strand = FALSE, strand_opp = FALSE,
  suffix = c(".x", ".y"), ...)

Arguments

x
tbl of intervals
y
tbl of intervals
invert
report `x` intervals not in `y`
strand
intersect intervals on same strand
strand_opp
intersect intervals on opposite strands
suffix
colname suffixes in output
...
extra arguments (not used)

Description

Identify intersecting intervals.

Note

Book-ended intervals have .overlap values of 0 in the output.

Examples

x <- tibble::tribble(
~chrom, ~start, ~end,
"chr1", 100,    500,
"chr2", 200,    400,
"chr2", 300,    500,
"chr2", 800,    900
)

y <- tibble::tribble(
~chrom, ~start, ~end, ~value,
"chr1", 150,    400,  100,
"chr1", 500,    550,  100,
"chr2", 230,    430,  200,
"chr2", 350,    430,  300
)

bed_intersect(x, y)
