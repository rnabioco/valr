Cluster neighboring intervals.

Usage

bed_cluster(x, max_dist = 0, strand = FALSE)

Arguments

x
tbl of intervals
max_dist
maximum distance between clustered intervals.
strand
cluster features on same strand

Value

data_frame

Description

Returned data_frame contains a new .id column that can be used for grouping along with chrom. Default max_dist of 0 means that overlapping and book-ended intervals will be clustered.

Examples

x <- tibble::tribble(
 ~chrom, ~start, ~end,
 "chr1", 100,  200,
 "chr1", 180,  250,
 "chr1", 250,  500,
 "chr1", 501,  1000
)

bed_cluster(x)
