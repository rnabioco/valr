# Provide working directory for valr example files.

Provide working directory for valr example files.

## Usage

``` r
valr_example(path)
```

## Arguments

- path:

  path to file

## Examples

``` r
valr_example("hg19.chrom.sizes.gz")
#> [1] "/home/runner/work/_temp/Library/valr/extdata/hg19.chrom.sizes.gz"
```
